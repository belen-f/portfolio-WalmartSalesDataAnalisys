# 📊 Análisis de Ventas | Walmart Sales Analysis

"Walmart" es una de las mayores cadenas minoristas del mundo.

## Descripción del proyecto

Análisis con base de datos proveniente del sitio web "Kaggle" sobre registros históricos de ventas semanales en diferentes tiendas de Walmart en EE.UU.

## Acerca del DataSet

Fuente: https://www.kaggle.com/datasets/mikhail1681/walmart-sales

"El conjunto de datos se creó con fines educativos y, en el momento de su creación no tenía interés en su origen" aclara el autor. Por lo que la veracidad de los datos no está confirmada. El uso que aquí se le da es con fines prácticos/educativos.

## Objetivo del análisis :

Investigar qué factores influyen en sus ingresos:

- Observar cómo cambian las ventas a lo largo del tiempo.
- Analizar las ventas por tienda.

## Resultados principales

- La tienda con mayores ventas fue la Store 20
- Las ventas aumentan en diciembre (Navidad)

## Tecnologías usadas

- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
- Kaggle datasets
